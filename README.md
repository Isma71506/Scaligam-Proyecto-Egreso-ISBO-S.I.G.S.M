# Scaligam-Proyecto-Egreso-ISBO-S.I.G.S.M

Proyecto Egreso BT Tecnologías de la Información 2026 — I.S.B.O.

Descripción

Sistema desarrollado para el Hospital de Clínicas, compuesto por dos módulos a integrar en su panel principal.

Módulo de gestión y digitalización de documentos para pacientes: permite a los funcionarios administrativos cargar informes y documentación médica por medio de códigos QR. Incluye un submódulo sobre encuestas de satisfacción.

Módulo de Ambulancias/Traslados: gestiona y da seguimiento a los traslados por parte de ambulancias del hospital y de otros centros médicos que aportan, registrando información sobre el conductor, enfermero, paciente, estado del traslado, etc.

Tecnologías utilizadas

Backend: PHP
Base de datos: MySQL / MariaDB 10.4.32
Frontend: HTML, CSS, JavaScript, Bootstrap 5.3
Servidor local de desarrollo: XAMPP 8.x
Control de versiones: Git + GitHub
Editor de código: Visual Studio Code

Instalación y ejecución local

1. Clonar el repositorio con git clone https://github.com/Isma71506/Scaligam-Proyecto-Egreso-ISBO-S.I.G.S.M

2. Abrir la carpeta del proyecto en Visual Studio Code

3. Abrir cualquier archivo .html dentro de /src dando click derecho y "Open With Live Server", se debe tener instalada la extensión Live Server

Aclaración: Esa es la manera de ejecutar localmente lo que se tiene hasta el momento para la primera entrega, se profundizará en XAMPP y demás cuando se tenga la base de datos para entregas posteriores.

Convenciones de commits

feat: se agrega una funcionalidad o pantalla. Ejemplo: feat: agrega formulario de alta de traslado
fix: se corrige un error
style: cambios de estilos o diseño sin afectar lógica. Ejemplo: style: ajusta colores del panel de ambulancias
docs: cambios en documentación. Ejemplo: docs: actualiza README con instrucciones de instalación
refactor: reorganización de código sin cambiar comportamiento. Ejemplo: refactor: reorganiza estructura de carpetas CSS
